Norwegian Electricity Spot Price Analysis

Time-Series Analysis (2021-2026)

Analysis Overview

This project investigates the volatile electricity market in Norway, focusing on price divergence between the North and South regions. Using Python for ETL and Power BI for visualization, we've mapped seasonal patterns, volatility, and price convergence factors.

Comparison across all 5 bidding zones (NO1-NO5).

Insights into grid upgrades, renewable output, and interconnector exposure.
