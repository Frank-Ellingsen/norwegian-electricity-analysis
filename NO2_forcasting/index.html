⚡ NO2 Electricity Price Forecast Dashboard

This page provides access to the interactive Streamlit dashboard for NO2 Electricity Price Forecasting.

View the Live Dashboard

To view the full interactive dashboard, please ensure the Streamlit application is running.

You can start it by navigating to the project root in your terminal and running:

streamlit run dashboard/app.py

Then, click the link below or access it directly at http://localhost:8501 (default address).

Open NO2 Forecast Dashboard

Note: The embedded dashboard below will only work if the Streamlit app is running on http://localhost:8501 and accessible from your browser.
